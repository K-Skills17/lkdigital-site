Product Requirements Document (PRD): LK Digital V1
1. Project Overview
Project Name: LK Digital - Dental Growth Flagship

Objective: Create a premium, luxury boutique lead-generation site for dental marketing.

Target Audience: High-end dental clinic owners in Portuguese-speaking markets.

Core Value Prop: 6 years of digital marketing excellence applied to the dental niche.

2. Technical Stack (AI-Optimized)
Framework: Next.js (App Router) for SSR (critical for SEO/GEO).

Styling: Tailwind CSS (for clean, maintainable utility classes).

Language: TypeScript (Strict mode) to ensure the codebase is robust for the future App transition.

SEO/GEO: Metadata API, JSON-LD Schema, and llms.txt implementation.

3. Aesthetic & UI Requirements
Theme: Luxury Boutique / Light Mode.

Palette: Background #FAFAFA, Text #1A1A1A, Accents #C5A368 (Gold).

Typography: Serif for headings (Cormorant Garamond), Sans-Serif for body (Inter).

Feel: Minimalist, spacious, high-end, stable. Avoid trendy animations; use subtle transitions.

4. Feature Scope (V1)
Multi-page structure: Home, About, Services, Contact, Blog (Marketing Dental), Legal.

Lead Gen: High-intent multi-step inquiry form.

Architecture: Modular component-based design to support future "Dashboard" features.
